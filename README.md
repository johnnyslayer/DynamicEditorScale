# DynamicEditorScale

# Installing the Plugin
Open Epic Games Launcher, head to Fab Library tab and find Dynamic Editor Scale Plugin. Press Install to Engine button and choose the appropriate engine version for you.

Open your project the engine version of which have the plugin already installed and head to Plugins tab. Find the Dynamic Editor Scale Plugin and set it enabled by checkbox. The engine will prompt to restart itself. Restart the editor and you will have plugin enabled and ready to work.

# How to Use:
When you set the plugin, as default you can use alt + mouse wheel to zoom in or zoom out the editor UI scale

Set tool Parameters in Editor Preferences / Plugin / Dynamic Editor Scale
